**Portfolio Dev** é um desafio prático desenvolvido durante os estudos na **Rocketseat**, com o objetivo de criar um portfólio moderno em HTML e CSS, apresentando identidade como desenvolvedor(a), principais habilidades e projetos em destaque.

## 🚀 Funcionalidades:

- Seção de apresentação personalizada
- Tags de tecnologias que identificam a stack
- Cards de projetos com imagem, título e descrição
- Layout moderno e responsivo
- Estrutura semântica com HTML
- Design focado em identidade visual profissional
